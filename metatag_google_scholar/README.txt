Metatag: Google Scholar
----------------
This module adds support for meta tag configuration for Google Scholar meta tags.

The following Google Scholar tags are provided:

    'citation_title'
    'citation_author'
    'citation_publication_date'
    'citation_journal_title'
    'citation_volume'
    'citation_issue'
    'citation_firstpage'
    'citation_lastpage'
    'citation_pdf_url'



Usage
--------------------------------------------------------------------------------
You can set any patterns for each tag so content type's field matches to each metatag of Google Scholar index.

Installation
--------------------------------------------------------------------------------
Enable the module after you enable Metatag.
